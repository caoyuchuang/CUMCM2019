![](http://img.candobear.com/2019-07-22-040532.png)

然后 Commit Changes：

![](http://img.candobear.com/2019-07-22-040735.png)

### 推送到原作者的仓库

合并之后，到自己的仓库会发现显示：This branch is X commits ahead of tansongchen:master，这个时候就点击 New pull request，
